Estou estudando banco de dados na faculdade e aprofundando meus conhecimentos tanto na parte teórica quanto prática, com foco em como as informações são estruturadas, armazenadas, manipuladas e utilizadas em sistemas reais. Esse aprendizado é fundamental para o desenvolvimento de aplicações mais organizadas, eficientes e profissionais.

Na parte teórica, estou aprendendo modelagem de dados, incluindo a criação de entidades, atributos e relacionamentos, além de entender como transformar problemas do mundo real em estruturas lógicas. Também estudo conceitos importantes como chaves primárias e estrangeiras, integridade referencial, cardinalidade e normalização, que ajudam a evitar redundância e melhorar a organização das informações dentro de um banco de dados.

Outro ponto importante é o entendimento de como os bancos de dados funcionam na prática, incluindo operações como inserção, atualização, exclusão e consulta de dados. Isso me ajuda a compreender como sistemas reais, como aplicativos bancários, sistemas de estoque e plataformas web, lidam com grandes volumes de informações de forma segura e eficiente.

Na parte prática, estou aplicando esses conhecimentos em projetos de programação, desenvolvendo sistemas como banco digital, sistema de posto de gasolina e biblioteca. Nesses projetos, preciso estruturar bem os dados, organizar regras de negócio e garantir que todas as operações funcionem corretamente. Isso me ajuda a consolidar a lógica de programação junto com a organização de dados.

Também estou começando a me preparar para o uso de bancos de dados relacionais como MySQL e PostgreSQL, entendendo como integrar esses bancos com linguagens de programação como Python. Isso abre caminho para o desenvolvimento de sistemas mais avançados, como aplicações web, APIs e sistemas corporativos.

Esses estudos estão sendo muito importantes para meu crescimento na área de tecnologia, pois me permitem entender não apenas como programar, mas como pensar de forma estruturada na construção de sistemas completos, organizados e escaláveis, semelhantes aos utilizados no mercado profissional.
